# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please do not open a public GitHub issue.

Instead, report it privately by emailing:
[security@keyvox.app](mailto:security@keyvox.app)

Please include:
- A description of the issue
- Steps to reproduce
- Affected version

Since KeyVox is a local-first application with no network services, most issues are likely to be low risk, but responsible disclosure is appreciated.
